When activate functions trail() and show(), first activate trail() and then show().

On key 'P' pause the game

On key 'S' slow down time, on key 'F' speed up time

On key 'N' reset time

When pause is pressed you can move planets around with mouse.

When pause is pressed, if you press ctrl and dragg planet, you change its velocity

Double click on planet to set it as center planet.

If more trail are turn on on the same time, it might drop fps
